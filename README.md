# README
Very basic tweet management system.

# Specs
* Ruby: 2.5.3
* Rails: 6.0.0.rc2
* Database: mySql
* jQuery 
* Bootstrap

# Setup instructions
* `git clone https://github.com/awlad/tweet.git`
* Run `bundle`
* Run `rake db:create && rake db:migrate`
* `rails s`
* Open browser & go to `localhost:3000`
